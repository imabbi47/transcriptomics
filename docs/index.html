🧬 Transcriptomics

RNA-seq analysis straight from a GEO/SRA accession number — for any organism.

What it does

Give it an accession (GSE…, GSM…, SRR…, SRP…) and it fetches the sequencing data and runs a full RNA-seq workflow — quality control, quantification, differential expression, and functional enrichment — ending in a single report.

The pipeline

Accession→ QC + Trim→ Quantify→ Diff. Expression→ Enrichment→ Report

Try it

Quick start (command line)

The engine works today from the terminal:

conda env create -f environment.yml
conda activate transcriptomics
pip install -e .

# fetch a whole GEO series as analysis-ready FASTQ + samplesheet
transcriptomics fetch GSE52778 -o data
